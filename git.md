# Работа с терминалом

## Команды для навигации в директориях ОС, и работы с файлами

+ ls - список всех файлов и папок в данной директории
+ mkdir - команда для создания новой директории
+ cd "имя директории" - имя папки можно указать из несколько букв и продолжить нажав TAB (имя заполниться автоматически)
+ cd .. - команда для перехода выше в директорию
+ cd ..//.. - команда для перехода на 2 директории выше
+ cd~ - команда для перехода в корневую директорию
+ touch - команда для создания файла

Например: touch Test.txt

+ cp - команда для копирования файла

Например: cp Test.txt test_cooy.txt

+ mv - команда для переименовывания файла

Например: cp test_cooy.txt test_copy.txt

+ **echo** "hello" > test.txt - команда для записи строки в нужный нам файл

+ **cat** test.txt - команда для просмотра содержимого файла

+ **rm** test.txt - команда для удаления файла или директории
